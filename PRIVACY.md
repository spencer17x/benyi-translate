# 本译隐私政策

生效日期：2026 年 7 月 14 日

本译是一款使用 Chrome 本地语言能力完成网页翻译的浏览器扩展。本政策说明本译如何处理用户数据。

## 数据处理范围

当用户主动点击扩展图标、使用快捷键、点击“开始翻译”或使用划词翻译时，本译会读取当前标签页中相应的可见网页文本或选中文本，用于在用户设备上检测语言并生成译文。网页地址和标签页标识仅用于维持当前翻译任务的连接与状态。

本译不会收集、上传、出售或向第三方共享网页正文、网页地址、译文、浏览记录、身份信息或使用行为。项目不包含账号系统、广告、分析、遥测或远程翻译服务。

## 本地处理与保存

- 网页正文和译文只在用户设备上处理。
- 翻译结果与任务状态仅保存在当前页面的内存中，并在页面关闭、导航或扩展重新加载后释放。
- 本译不会保存完整网页快照，也不会将网页内容写入持久化存储。
- 首次使用某个语言对时，Chrome 可能下载本地语言资源；该过程由 Chrome 管理，本译不会把待翻译网页正文发送给语言资源服务。

## 权限用途

- `activeTab`：仅在用户主动触发后访问当前标签页。
- `contextMenus`：在用户选中文本时提供“使用本译翻译选中文本”的右键入口。
- `offscreen`：在不打开窗口、标签页或侧边栏的情况下运行 Chrome 本地 Translator API；正文不会因此离开设备。
- `scripting`：按需向当前页面注入翻译和双语展示逻辑。
- `sidePanel`：提供可选的翻译控制、进度、显示模式和撤销入口；关闭后翻译仍会继续。

本译只申请当前功能需要的最小权限，不申请后台持续访问所有网站的权限。

## 数据安全与有限使用

译文以纯文本形式写入页面，不执行网页内容或译文中的代码。本译不会在运行时下载并执行远程 JavaScript。

通过 Chrome API 获得的信息仅用于提供用户主动请求的本地网页翻译功能。本译对这些信息的使用遵守 Chrome Web Store User Data Policy，包括 Limited Use 要求。

## 政策变更

如果本译未来改变数据处理方式，本政策会在相关版本发布前更新，并在扩展界面或商店页面中明确说明。

## 联系方式

如有隐私或安全问题，请通过项目的 [GitHub Security](https://github.com/spencer17x/benyi-translate/security) 私下报告；一般问题可在 [GitHub Issues](https://github.com/spencer17x/benyi-translate/issues) 提交。
