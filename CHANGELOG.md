## [0.1.0](https://github.com/spencer17x/benyi-translate/releases/tag/v0.1.0) (2026-07-14)

### Features

- 使用 Chrome 本地语言能力完成英语网页到简体中文的翻译。
- 支持原文、双语和仅译文显示，以及暂停、取消、重新翻译和撤销。
- 支持 X 帖子正文识别和按需页面注入。
- 支持 `Ctrl+Shift+Y`；macOS 使用 `Command+Shift+Y`。
- 提供正式扩展图标、窄屏 Side Panel 和深色模式。

### Privacy

- 网页正文和译文仅在用户设备上处理，不上传到远程翻译服务。
