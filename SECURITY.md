# 隐私与安全

本译的核心承诺是：默认在用户设备上完成语言识别和翻译，不把网页正文、选中文本或译文发送到远程翻译服务。

## 数据边界

扩展可以在用户授权的页面中读取可见文本，以执行翻译。首版设计要求：

- 不上传网页正文和译文。
- 不保存完整网页快照。
- 不默认启用分析、遥测或行为追踪。
- 首版翻译缓存仅存在于页面内存并随页面生命周期释放。
- 公开测试版本的持久化翻译缓存默认关闭；用户明确启用后可查看占用、关闭并一键清除。
- 调试信息只包含版本、能力状态、数量、耗时和错误码，不包含完整正文。

## 权限原则

- 默认只申请完成当前功能所需的最小权限。
- 持久站点访问通过可选权限按需申请。
- 新增权限必须在变更说明和用户文档中解释用途。

## 安全实现要求

- 译文作为纯文本写入，不解析或执行译文 HTML。
- 划词浮层使用隔离的 Shadow DOM，选中文本和译文均通过纯文本节点展示。
- 不执行来自网页的代码。
- 不在运行时下载并执行远程 JavaScript。
- 校验所有跨组件消息和来自网页的数据。
- 页面导航或标签关闭后中止相关任务并丢弃迟到结果。
- 整页翻译运行在扩展自带的隐藏文档中，仅通过经过校验的扩展消息与当前标签页通信。

## 报告安全问题

请不要为潜在漏洞创建公开 Issue。请使用 GitHub 仓库的 **Security → Report a vulnerability** 私下提交报告，并提供：

- 问题类型和影响范围。
- 最小复现步骤或验证代码。
- 受影响版本和环境。
- 建议修复方式（如有）。

请勿在报告中附带与问题无关的私人网页内容。
